# qfield-filter-plugin
Плагин для фильтрации в qfild. Аналог SQL запросов к слоям.
Можно применять к нескольким слоям.
Доступны опреаторы "=", "!=", "<", ">", "<=", ">=", "LIKE", "ILIKE". 
Объеденение или пересенчение услвоий.
Доступен список отфильтрованных объектов.
Все остальные объекты, удаляются со слоя, пока не измениться условия или не отменити фильтрацию.
